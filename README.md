### Front End React vite
* Criando projeto
    ```sh
    npm create vite@latest
    ```
* Instalando pacotes
    ```sh
    npm install
    ```
* Criando projeto
    ```sh
    npm run dev
    ```
* Styled-Components
    ```sh
    npm install --save styled-components@latest
    ```
    ```sh
    npm install --save @types/styled-components
    ```
